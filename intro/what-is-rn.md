## Quick Start

To get started immediately:

1. **Install Expo CLI**
   ```bash
   npm install -g @expo/cli
   ```

2. **Create a new project**
   ```bash
   npx create-expo-app MyFirstApp
   cd MyFirstApp
   npm start
   ```

3. **Install Expo Go** on your phone and scan the QR code
